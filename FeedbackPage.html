Feedback

1. Which of the following is the correct format for the CSS?

Correct Answer: p { color: red; text-align: center; }

Feedback: CSS syntax is composed of selector and declaration wherein selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons.Each declaration includes a CSS property name and a value, separated by a colon and declaration blocks are surrounded by curly braces.

See W3Schools for more information.

2. Which of the following examples is the correct way to use CSS ID selector?

Correct Answer: #sen1 { text-align: center; color: red; }

Feedback: Hash (#) character is being used to select an element with a specific id. An ID name cannot start with a number.

See W3Schools for more information.

3. Which of the following can be a possible value for margin properties?

Correct Answer: auto and %

Feedback: auto and % do not exist as values in CSS margin properties.

See W3Schools for more information.

4. What should be, so the width of a table that is to be adjusted to the current width of the browser window?

Correct Answer: 100%

Feedback: Table width is identified in percentages(%). The width value of 100% indicates a width for the table that is the full width of the browser window.

See W3Schools for more information.

5. To wrap a block of text around an image, the appropriate css property to use is?

Correct Answer: Float Property

Feedback: The float property in CSS is used for positioning and layout on web pages. A common usage might be floating an image to one side and letting text wrap around it.

See W3Schools for more information.
